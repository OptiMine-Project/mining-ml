# Dataset Needs

### Dataset Mine-Master (dummy)
Informasi untuk tambang-tambang yang dimiliki perusahaan 

### Dataset Weather Data
Dataset riwayat cuaca untuk setiap lokasi tambang. Didapatkan dari satelit
Sumber: ERA5 dataset https://cds.climate.copernicus.eu/datasets/reanalysis-era5-single-levels?tab=download
Diproses berdasarkan kebutuhan project

### Dataset Equipment Inventory (dummy)
Dataset ini berisikan list perlengkapan dan alat berat yang digunakan dalam tambang

### Dataset Road Condition (dummy)
Dataset road condition berisikan rute jalan yang ditempuh alat angkut melalui jalur darat beserta keterangan kondisi jalan

### Dataset Effective Capacity (dummy)
Dataset ini berisikan data historis rencana penambangan setiap minggu sejak tambang digunakan. Dataset ini juga menyajikan pencapaian asli produksi tambang.

### Dataset Production Plan (dummy)
Dataset Effective Capacity merupakan dataset berisikan data historis kondisi alat tertentu yang menjadi faktor utama produksi dalam rencana produksi mingguan

### Dataset Production Constrains (dummy)
Dataset realisasi production plan dan kendala yang saat pelaksanaan kerja.

### Dataset Shipping Schedule (dummy) 
Dataset shipping schedule berisikan data historis jadwal pengiriman.
